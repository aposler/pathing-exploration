Pathing Algorithm Exploration

This page allows you to explore the difference between 4 pathing algorithms, A*, Breadth First, Depth First, and Greedy Best First. Click and drag on the graphs below to create obstructions, which the algorithms will avoid when creating their paths. Use the top slider to contol the grids dimensions. Use the Username and grid name controls to save your created grids and load them back later. Use the second slider to control the delay between pathing algorithm iterations, and the buttons below that to manually control the algorithms.
